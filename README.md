# Employee Leave Management System (Using MERN for gamage assignment)

Backend-focused Leave Management System where:
- Employees can apply for leave and view their own leave history
- Admin can view all leave requests and approve/reject them
- JWT authentication + role-based authorization
- MongoDB + Mongoose
- Bonus: Audit logs for admin actions

## Tech Stack
- Backend: Node.js, Express.js, MongoDB (Mongoose)
- Auth: JWT, bcrypt
- Frontend: React (minimal)
